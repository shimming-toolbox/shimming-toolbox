************
Installation
************

``Shimming Toolbox`` is written in Python. It can be used either from the Terminal
on all supported platforms (see below) or it can be used from a graphical user interface (GUI) as a plugin for
`FSLeyes <https://fsl.fmrib.ox.ac.uk/fsl/fslwiki/FSLeyes>`__ (macOS and Linux).

``Shimming Toolbox`` is supported on ``macOS``, ``Linux`` and ``Windows`` operating systems.
